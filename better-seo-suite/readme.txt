Comprehensive SEO plugin for Posts, Pages, WooCommerce, and EDD. Includes patterns, sitemaps, social, schema, breadcrumbs, and importer.

== Description ==
Better SEO Suite provides essential SEO features: custom titles and descriptions with patterns, XML sitemaps, robots integration, OpenGraph and Twitter Cards, JSON-LD schema for Articles and Products, WooCommerce and Easy Digital Downloads integrations, breadcrumbs, and data importers for Yoast, All in One SEO, and Rank Math.

== Installation ==
1. Upload the `better-seo-suite` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Better SEO in the admin sidebar to configure settings.

== Frequently Asked Questions ==
Q: Can I import from Yoast?
A: Yes. Go to Better SEO > Tools and run the import.
